# 第1章 Qt Quick概览

高度直观的Qt API为C++编码，

快速上手的Qt标记性语言QML。

同时使用它们：为后端应用逻辑使用Qt C++，而使用QML开发用户界面

qml，语法格式非常像CSS，但又支持JavaScript形式的编程控制

在QML中，通过具有属性的对象树来表示用户界面

Qt Quick实现了ECMAScript语言，它作为QML的脚本语言，可以对QML进行逻辑方面的编程
